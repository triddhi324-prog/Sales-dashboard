# Sales-dashboard
Sales Intelligence Dashboard using Python, SQL &amp; Power BI

 Project Overview

This project is an end-to-end analysis of retail sales data aimed at understanding how a business performs across regions, products, and customer segments.

The goal was not just to build charts, but to answer real business questions like:

* Where is the company making the most money?
* Which products are actually profitable?
* Who are the most valuable customers?

Business Problem
Retail businesses often have large amounts of sales data but struggle to turn it into clear insights.

Key challenges include:

* Tracking revenue across different regions
* Understanding customer purchasing behavior
* Identifying profit-driving vs loss-making products

This project focuses on solving these problems using a structured data analysis approach.

 Tech Stack

* Python (Pandas) → Data cleaning & feature engineering
* MySQL → Data querying & aggregation
* Power BI → Dashboard & visualization
* Excel → Initial data exploration

 What I Did

* Cleaned and transformed raw sales data using Python
* Created new metrics like **Profit Margin** and **Monthly Sales Trends**
* Wrote SQL queries to identify top customers and category performance
* Built an interactive Power BI dashboard with filters and KPIs

 Key Insights

* The **West region** consistently generates the highest revenue
* **Electronics** is the top-selling category but also has higher discounts impacting profit
* A small percentage of customers contribute a large share of total sales (Pareto effect)
* Some products have high sales but low profitability, indicating pricing or discount issues

 Dashboard Highlights

The Power BI dashboard includes:

* KPI cards for **Total Sales, Profit, and Profit Margin**
* Monthly sales trend to track performance over time
* Regional analysis to compare performance across locations
* Customer segmentation to understand different buyer groups
* Interactive slicers for dynamic filtering

 Skills Demonstrated

* Data visualization and dashboard design
* Translating data into business insights
